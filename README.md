# todoComReact

1. Faça o Git clone.

2. Instale o "npm i" na pasta Backend e Frontend.

3. De início no projeto com "npm run dev" no Backend e Frontend.

4. Backend rodará na porta http://localhost:3003 e Frontend na porta http://localhost:8080.

5. Com a aplicação rodando pode criar, editar, deletar e pesquisar.

6. Na barra de pesquisa temos os atalhos "Shift + Enter" para buscar pesquisa, "Esc" para apagar a pesquisa e "Enter para adicionar uma nova tarefa.
